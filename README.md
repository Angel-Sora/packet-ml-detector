# 🛡️ Packet ML Detector

### ML-based Network Anomaly Detection System

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux-lightgrey.svg)]()

**Packet ML Detector** — это приложение с графическим интерфейсом для обнаружения аномалий в сетевом трафике с использованием машинного обучения (Isolation Forest).

</div>

---

## ✨ Features

- 🔍 **Real-time packet capture** via Scapy/Npcap
- 🧠 **Anomaly detection** using Isolation Forest algorithm
- 🖥️ **User-friendly GUI** built with Tkinter
- 📊 **Interactive dashboard** powered by Streamlit
- 📝 **Alert logging** to JSON format
- 🧹 **One-click cleanup** of logs and cache
- ⚙️ **Easy interface configuration**

---

## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher
- Npcap (Windows) or libpcap (Linux)

### Installation

```bash
# Clone the repository
git clone https://github.com/Angel-Sora/packet-ml-detector.git
cd packet-ml-detector

# Install dependencies
pip install -r requirements.txt
```
